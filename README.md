# 1C-Trade-Automation
Внешняя обработка для 1С 8.3: Загрузка прайсов, сверка остатков и управление заказами.
# Инструмент автоматизации торговли (1С 8.3)

Внешняя обработка для автоматизации работы менеджера по закупкам. Проект демонстрирует навыки работы с запросами, временными таблицами и объектной моделью 1С.

### Основной функционал:
*   **Загрузка данных:** Чтение внешних файлов (Excel/CSV) и преобразование в коллекции 1С.
*   **Интеллектуальная сверка:** Сопоставление данных файла с базой через запросы к справочникам и регистрам сведений.
*   **Автоматизация документов:** Программное создание документов «Заказ поставщику» на основе анализа отклонений цен.

### Технический стек:
*   1С:Предприятие 8.3 (Управляемые формы).
*   Язык запросов 1С (Временные таблицы, СрезПоследних, Соединения).
*   Стандарты разработки фирмы «1С».

### Структура проекта:
*   `/src` — исходный код модулей в текстовом формате (.bsl).
*   `/bin` — готовый файл внешней обработки (.epf) для тестирования.
